mel

A tiny CLI that makes a few Git workflows feel friendly.

start save deploy freshstart status pull test

Install

curl -L -o /usr/local/bin/mel https://raw.githubusercontent.com/your/repo/main/mel
chmod +x /usr/local/bin/mel

Quick start

mel start           # prompts for a name if omitted
# edit files...
mel save           # commit, rebase, push
mel status         # see what's going on
mel deploy         # run tests, fast-forward merge into main

Configure tests

Create .mel/config.json in your repo root and add commands to run before deploy or when invoking mel test:

{
  "test_commands": [
    "npm ci && npm test -s",
    "pytest -q --disable-warnings"
  ]
}
